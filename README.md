# Predictive Analytics Using Historical Data

## Project Overview

This project focuses on analyzing historical sales data to identify patterns, trends, and business insights. Using data analytics and machine learning techniques, future sales are forecasted to support data-driven decision-making.

## Objectives

- Analyze historical sales performance.
- Identify sales and revenue trends.
- Perform data cleaning and preprocessing.
- Build predictive models for sales forecasting.
- Create an interactive Power BI dashboard.
- Generate actionable business insights.

## Dataset Features

The dataset contains the following columns:

- Date
- Product_ID
- Product_Category
- Sales
- Revenue

## Tools & Technologies

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook
- Power BI

## Project Workflow

1. Data Collection

Imported historical sales data from CSV files.

2. Data Cleaning

- Removed duplicate records.
- Handled missing values.
- Converted data types.
- Prepared data for analysis.

3. Exploratory Data Analysis (EDA)

- Sales trend analysis.
- Revenue trend analysis.
- Product category performance.
- Correlation analysis.

4. Feature Engineering

- Date-based features.
- Monthly and yearly aggregations.
- Sales growth calculations.

5. Predictive Modeling

- Built a Linear Regression model.
- Trained the model using historical sales data.
- Generated future sales predictions.

6. Dashboard Development

Created an interactive Power BI dashboard containing:

- Total Sales KPI
- Total Revenue KPI
- Average Sales KPI
- Sales Forecast Chart
- Revenue Trend Analysis
- Product Category Analysis
- Top Product Performance
- Sales Distribution Charts
- Interactive Filters and Slicers

## Key Insights

- Identified sales trends over time.
- Determined top-performing product categories.
- Forecasted future sales using historical patterns.
- Provided business recommendations based on data insights.

## Results

The predictive model successfully estimated future sales trends and helped visualize business performance through an interactive dashboard.

## Future Enhancements

- Implement advanced forecasting models.
- Add real-time data integration.
- Deploy dashboard to cloud platforms.
- Improve model accuracy using ensemble techniques.

## Conclusion

This project demonstrates the application of predictive analytics on historical sales data. By combining data analysis, machine learning, and visualization, businesses can make informed decisions and improve future planning.
